myOS — a hobby OS built from scratch.

  Phase 1 — Multiboot boot, VGA terminal
  Phase 2 — GDT, IDT, PIC, PS/2 keyboard
  Phase 3 — Physical memory manager, paging, kernel heap
  Phase 4 — Interactive shell
  Phase 5 — Virtual RAM disk (TAR), this filesystem
  Phase 6 — PIT timer, uptime, mem, reboot, halt, system info
